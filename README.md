# SimpleVPN
------------------------
To compile the compile:
------------------------

$ make

------------------------
To run the server:
------------------------
$ sudo ./vpnserver

------------------------
To run the client:
------------------------

First change the SEVER_IP in vpnclient.c to match with the server's ip,
$ sudo ./vpnclient server's domain
